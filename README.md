Road to 1.2億 (Asset Formation Simulator)
55歳までに1.2億円という目標を達成するための、個人向け資産形成シミュレーションアプリです。 単なる計算機ではなく、将来の資産推移を可視化することで、日々の積立のモチベーションを維持することを目的に開発しています。

# 🌟 主な機能
- 精密な複利計算ロジック: 月次での積立と複利を考慮した、精度の高い資産シミュレーション。
- インタラクティブなグラフ表示: Vicoライブラリを採用し、将来の資産推移を視覚的に把握可能。
- 期間切り替え機能: 「全期間」と「直近10年間」をワンタップで切り替え。初期の小さな資産変化もズームアップして確認できます。
- 目標差額の可視化: 最終目標（1.2億円）までの不足額を自動計算して表示。

# 🛠 使用技術
- 言語: Kotlin
- UI フレームワーク: Jetpack Compose (Modern Toolkit)
- グラフライブラリ: Vico
- アーキテクチャ: MVVM (Model-View-ViewModel) 指向
- デザイン: Material Design 3

# 🚀 こだわったポイント (エンジニアとしての工夫)
- State Hoisting による疎結合な設計: 入力フォームを別コンポーネントとして切り出し、状態管理を親に集約することで、コードの再利用性とテストのしやすさを向上させました。
- スケーリング課題の解決: 目標額が大きいために直近の変化がグラフで見えなくなる問題に対し、リスト操作（take関数）を用いた期間切り替え機能を実装し、UXを改善しました。
- 型の安全性の徹底: Kotlinのスマートキャストを活用し、!!（非NULL表明）を使わない安全なコードを記述することで、実行時エラー（NPE）を徹底的に排除しました。
